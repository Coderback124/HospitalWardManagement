# Hospital Ward Management

A C# console application for managing hospital wards, patient admissions, and doctor registrations. The application demonstrates object-oriented software design while providing an organised approach to managing hospital personnel and ward allocations.

---

## Features

- Register doctors
- Admit patients to hospital wards
- Manage multiple hospital wards
- Manage ward capacity
- Display registered doctors and admitted patients
- Object-oriented class hierarchy
- Inheritance
- Polymorphism
- Interface implementation
- Exception handling
- Console-based menu system

---

## Hospital Wards

The application manages the following wards:

- General Ward
- Paediatric Ward
- Intensive Care Ward

Each ward has a maximum capacity of four patients.
